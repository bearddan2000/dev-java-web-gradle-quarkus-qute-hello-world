# dev-java-web-gradle-quarkus-qute-hello-world

## Description
A POC for quarkus framework
using jsp-like template engine,
qute.

## Tech stack
- java
- gradle
  - quarkus
    - jsp

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
